# assembly
assignments for school in assembly

run `./m <path or name of asm file in src>`

ex. 
+ `./m src/01_math.asm`
+ `./m 01_math.asm`
+ `./m 01_math`

all are valid

##assignment 01 (01_math.asm):
reads an int and
produces
answer=A*B-(A+B)/(A-B)

##assignment 02 (02_average.asm):
reads n integers and reports their average
